Databáze

Instalace

Požadavky:
  • Laravel Framework 6.4.1
  • PHP 7.3.10 a MySQL server 8.0.18, nebo XAMPP 7.3.11
  • Composer 1.9.0
Postup:
  • Naklonujte projekt do cílového adresáře: git clone https://github.com/vamos/stunning-octo-carnival
  • Nainstalujte vše potřebné pro kompatibilní Homestead prostředí viz. požadavky výše
  • Přihlašte se do mysql: mysql -u root -p
  • Vytvořte databázi: CREATE DATABASE IIS;
  • Změňte heslo uživatele root na 'd6ed06', tak aby odpovídalo záznamu v souboru .env v adresáři Projekt. Nebo upravte položky .env : DB_DATABASE, DB_USERNAME a DB_PASSWORD podle vašeho mysql nastavení.
  • V kořenovém adresáři projektu
  • Nainstalujte composer: composer install
  • Spusťte migraci databáze: php artisan migrate:fresh --seed
  • Spusťte server: php artisan serve
  • Otevřete uvítací stránku v proklížeči na adrese: http://localhost:8000

Známé problémy

  • Vše je implementováno přesně podle zadání.
